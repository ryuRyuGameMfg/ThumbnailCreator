## ⚠️ 景品表示法チェック

### 🔴 重大な問題

#### 1. **「期間限定」の期間が明示されていない**

**問題箇所**: `Pricing.tsx` 28-29行目
```tsx
<div className="...">期間限定</div>
```

**問題点**:
- 景品表示法では、価格比較表示を行う際、割引価格が適用される期間を明確に表示する義務がある
- 「期間限定」だけでは、具体的な期間が不明確で、消費者が誤解する可能性がある

**法的リスク**:
- 不当表示（有利誤認表示）と判断される可能性
- 消費者庁からの措置命令や課徴金納付命令の対象となる可能性

**修正案**:
```tsx
// 案1: 具体的な期間を明示
<div className="...">期間限定（2024年12月31日まで）</div>

// 案2: 期間限定タグを削除し、価格比較のみ表示
// （通常価格と比較価格を表示するだけで十分な場合）

// 案3: 「初回限定」など、より具体的な表現に変更
<div className="...">初回限定価格</div>
```

#### 2. **価格比較表示の根拠**

**問題箇所**: `Pricing.tsx` 43行目
```tsx
<div className="text-[10px] text-gray-400 line-through">100,000円</div>
<div className="text-lg font-bold text-sky-600">75,000<span className="text-xs font-normal">円</span></div>
```

**確認事項**:
- ✅ 通常価格（100,000円）と割引価格（75,000円）が明確に表示されている
- ⚠️ ただし、「期間限定」の期間が不明確なため、上記の問題と関連

**推奨対応**:
- 通常価格の根拠（過去の販売実績など）を確認できる状態にしておく
- 期間限定の期間を明示する

