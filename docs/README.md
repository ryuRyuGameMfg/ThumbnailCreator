## クイックスタート

新しいスライドを作成する場合:

1. `docs/REGULATION.md` を読む
2. `src/constants/COLORS.ts` から色定数をインポート
3. `src/constants/STYLES.ts` からスタイル定数をインポート
4. `src/components/common/` の共通コンポーネントを活用
